# File Sharing Service

Сервис для загрузки файлов и получения уникальных ссылок на скачивание.

## Возможности

- Авторизация пользователей
- Загрузка файлов
- Генерация уникальных ссылок для скачивания
- Скачивание файлов по ссылке
- Статистика скачиваний
- Автоматическое удаление файлов, которые не скачивались более 30 дней

## Технологии

- Node.js
- Express
- Multer
- Vanilla JavaScript
- HTML/CSS

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

или

```bash
npm start
```

После запуска приложение будет доступно по адресу:

```text
http://localhost:3000
```

## Использование

### Авторизация

```text
http://localhost:3000/login.html
```

Для входа используйте учетные данные, указанные в файле `.env`.

### Загрузка файлов

```text
http://localhost:3000
```

### API

Авторизация:

```http
POST /login
```

Выход:

```http
POST /logout
```

Загрузка файла:

```http
POST /upload
```

Скачивание файла:

```http
GET /download/:id
```

Получение статистики:

```http
GET /files
```
